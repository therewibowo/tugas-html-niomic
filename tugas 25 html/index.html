    • Ketika kita ingin memvuat layout untuk menampilkan block artikel-artikel yang ada di website kita, tag article bisa kita gunakan dengan element header dan footer. Karena tag article memiliki banyak artinya element ini memiliki lebar sesuai lebar browser

    • Element ini kita gunakan untuk melengkapi layout utama.Misal kita memiliki halaman blog makan kita bisa menambhakan article. karena asice memiliki sifat bloack maka lebarnya sesuai page jika kita tidak menemukan lebarnya
